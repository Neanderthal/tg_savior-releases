# TG Savior — Downloads

В этом репозитории публикуются установщики **TG Savior** —
приложения для резервного копирования Telegram-данных на Windows.

Исходный код программы находится в приватном репозитории; здесь —
только скомпилированные релизы.

👉 **Последняя версия:** см. [Releases](../../releases/latest)

Каждая запись релиза содержит:

- `TG_Savior_Setup.exe` — установщик для Windows 10/11
- `TG_Savior_Setup.exe.sha256` — контрольная сумма для проверки целостности
- `updates.json` — манифест для будущей функции автообновления
